# Third-party notices

This project bundles, and is validated against, **Triangle** by Jonathan
Richard Shewchuk. The following artifacts are derived from Triangle:

- `src/main/resources/win32-x86-64/triangle.dll` — a compiled build of a
  modified, reduced (library-only) version of Triangle, used by
  `NativeTriangleMesher` via JNA;
- `src/test/resources/contract/predicates.txt` and
  `src/test/resources/meshes/*.txt` — data files generated by running Triangle.

The pure-Java mesher and the rest of the source in this repository are original
work; they implement published algorithms (Delaunay triangulation, Ruppert
refinement, Shewchuk's orientation/incircle predicates) and are validated
against Triangle's output, but are not derived from Triangle's source code.

The bundled `triangle.dll` is built from a **modified, reduced** copy of
Triangle (library-only; many unused features removed). The full modification
notice and the modified source live in the companion C reference repository;
see its `README` ("MODIFICATION NOTICE") and `README-WINDOWS.md`. The bundled
binary is provided free of charge for non-commercial use.
