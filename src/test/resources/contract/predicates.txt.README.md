# Predicate oracle

`predicates.txt` is the cross-language contract for the robust geometric
predicates. Each line is one test case:

```
orient2d  ax ay bx by cx cy        <sign>
incircle  ax ay bx by cx cy dx dy  <sign>
```

`<sign>` is `-1`, `0`, or `+1` — the **exact** sign of the orient2d / incircle
determinant for those coordinates. Coordinates are written at full `%.17g`
precision, so they parse back to the identical `double`. Only the sign matters;
a correct robust predicate may compute a different magnitude but must reproduce
every sign here, including the near-degenerate cases that defeat naive `double`.

## Provenance

Generated by the C reference implementation:

- repo: `triangle` (the reduced, library-only Triangle)
- generator: `test/predicate_oracle.c`
- source of truth: `test/golden/predicates.txt`

If the C predicates ever change, regenerate there (`pwsh -File
test/run-tests.ps1 -Update`) and re-copy the file here. Do not edit by hand.
